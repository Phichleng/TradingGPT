# ICT Playbook (sample)
Bullish model: during the London or NY-AM killzone, price sweeps a prior
sellside liquidity pool (Asian low / equal lows), prints a displacement leg
leaving a bullish FVG, then retraces into the 0.62-0.79 OTE which should overlap
a bullish order block in discount. Entry on the OTE/OB confluence; stop below the
swept low; target the opposing buyside liquidity. Invalidate if no sweep precedes
the move, or if entry would be in premium against bullish HTF bias.
